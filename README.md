# Plumby - Professional Plumbing Service Website

A modern, responsive website for a professional plumbing service built with Astro, Tailwind CSS, and Font Awesome.

## 🚀 Tech Stack

- **Framework:** Astro
- **Styling:** Tailwind CSS
- **Icons:** Font Awesome
- **Fonts:** Inter, Satoshi

## 🛠️ Development

### Prerequisites

- Node.js (v16 or higher)
- npm (v7 or higher)

### Setup

1. Clone the repository
```bash
git clone [repository-url]
cd plumber
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

The site will be available at `http://localhost:4321`

## 🎨 Design System

### Colors
- Primary Blue: `#1681FF`

### Typography
- Primary Font: Inter (400, 500, 600)
- Logo Font: Satoshi (700)

## 🚀 Deployment

Build the project for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
